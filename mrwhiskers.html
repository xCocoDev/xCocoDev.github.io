Mr. Whiskers

The game is a lane-based runner about a cat's journey to heaven made with Unity. It features a story mode as well as an endless one. Some of the mechanics that the player encounters throughout the game are: enemies, moving platforms, portal connections to other levels, power-ups.

Footage

Interesting things that I did

  • Persistent Leaderboard system with an arcade style controller view.
  • Player Movement system with smooth transitions and animations.
  • Player Tracking system to record player's decisions (Explorer, Killer, Achiever, Socializer).
  • Endless Levels spawning and shuffling system.
  • Player Interaction systems with the environment objects (collectables, sounds, particle effects).
  • A Ghost Spawn system to see where other players died in earlier runs.
