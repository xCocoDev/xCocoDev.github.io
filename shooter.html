Multiplayer TPS

A Multiplayer Third Person Shooter demo project currently in-progress made only by me in Unreal Engine 4. It's sole purpose is to showcase some of my C++ and UE4 skills.

The core game mechanic is that players could team-up or play solo and battle against some advanced AI's with an ever increasing in difficulty wave spawn system.

Screenshot

Interesting things that I did

  • Complex AI Behavior Tree and EQS that features: Finding Cover and Shooting, Fleeing to Cover when Low Health to Regenerate.
  • Wave Spawn System that uses EQS to find the best spawning location for AI's.
  • Character Controller for Players and AI's that has full movement animations using Locomotion.
  • Extendable Weapon System with a shooting mechanic, right now I implemented only two types: Rifle and Grenade Launcher.
  • All the gameplay systems are replicated, so that the game can be played in Multiplayer.
  • A Power Up system that can be easily extended in order to create other types of Power Ups.
  • Player Interaction systems with the environment objects (collectables, sounds, particle effects).
