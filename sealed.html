Sealed

A puzzle game made in a Custom C++ game engine. The core gameplay mechanic is that the player may give up arrow keys (is not able to use it anymore to move) to create panels and use them to solve the puzzles. All in-game systems are scripted by me using the Lua Scripting API that is embedded in the engine.

Footage

Interesting things that I did

  • All gameplay systems such as: Player Controller, Level Manager, Interactable Behaviors.
  • Game State serialization system.
  • Level Importer after the level layout file has been exported from Unity.
  • Coordinating UI programmer.
  • Grid system to spawn the game objects on after the level layout was build in Unity.
  • Player Interaction systems with the environment objects (collectables, sounds, particle effects).
